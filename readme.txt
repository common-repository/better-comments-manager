Better Comments manager allows you to view your comments post wise, it also allows you to reply to your comments from within admin panel.

== Description ==

Better comments manager is a plugin which allows you to reply to comment from within the admin panel. It uses Ajax to respond to comments so you do not have to leave the comments page while commenting.

The latest version of Better comments manager has support to edit the comments from within the admin panel too. The plugin is also compatible with brian's threaded comments. 




== Installation ==

To do a new installation of the plugin, please follow these steps

1. Download the better-comments-manager.zip file to your local machine.
2. Unzip the file 
3. Upload `better-comments-manager` folder to the `/wp-content/plugins/` directory
4. Activate the plugin through the 'Plugins' menu in WordPress

If you are upgrading from previous versions please follow these steps

1. Download the latest version of better comments manager
2. De-activate the better comments manager plugin
3. Delete the directory containing the plugin files
4. Unzip the file 
5. Upload `better-comments-manager` folder to the `/wp-content/plugins/` directory
6. Activate the plugin through the 'Plugins' menu in WordPress


== Frequently Asked Questions ==

= What is better comments manager =

Better comments manager is a plugin that allows you to instantly reply to comments from within the admin panel. It also allows you to sort comments from within the admin panel

= Is Better comments manager compatible with Brian's Threaded Comment? =

Uptil version 1.2 there is no support for Brian's threaded comment, adding a reply using better comments manager will add a new comment. The compatibility will be released in version 1.3
